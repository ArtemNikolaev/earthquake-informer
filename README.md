# Earthquake Informer
Shows on the world map the points where the earthquake occurred today.

[Website](https://artemnikolaev.github.io/earthquake-informer/)

---
# how to run

_NodeJS_: >=14

```
npm install

npm start // to serve development
npm run build // to build production
```
